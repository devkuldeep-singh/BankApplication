<h1> Bank Management System</h1>

A web-based Bank Management System developed using Java and Advanced Java technologies.
This application simulates core banking operations such as account management, secure authentication, money transfer, loan requests, and transaction history tracking.<br>

<h3>Features</h3>

Secure user registration and login<br>

Account creation and management<br>

Money transfer between accounts<br>

Loan request and processing module<br>
Transaction history and mini-statement<br>

 Web-based interface with dynamic pages<br>

<h3> Technologies Used<h3>

Backend: Java, Advanced Java (Servlets, JSP, JDBC)<br>

Frontend: HTML, CSS, JavaScript<br>

Database: Oracle SQL<br>

Server: Apache Tomcat<br>

IDE: Eclipse <br>

 <h3>Project Architecture</h3>

MVC-based web application<br>

Servlets handle business logic<br>

JSP used for dynamic UI rendering<br>

JDBC used for database connectivity<br>

Oracle SQL used for data storage and retrieval<br>
